API Endpoints

Generates a password with a fixed length of 16, using special and alphanumeric characters:
api/v1/password

Generates a password with variable length (min: 8), using special and alphanumeric characters:
api/v1/password/length Example: api/v1/password/length?length=10 Note: The maximum length for this type of password is 92

Generates a 16-character password using only alphanumeric characters:
api/v1/password/alphaNum

Generates a variable-length alphanumeric password (min: 8):
api/v1/password/alphaNum/length Example: api/v1/password/alphaNum/length?length=10 The maximum length for this type of password is 62

Generates num passwords (special + alphanumeric):
api/v1/passwords/num Example: api/v1/passwords/num?num=4

Note: num > 1 and < 16

Generates num passwords with a custom length (special + alphanumeric):
api/v1/passwords/num/length Example: api/v1/passwords/num/length?num=4&length=12

Note: num > 1 and num < 16 and length > 7 and length < 93.
